## 5. 최종 검증

가능한 경우 아래 명령을 모두 통과해야 한다.

```bash
npm run lint
npm run test
```

프로젝트 초기화 전이라 명령이 없다면, package scripts를 만든 뒤 다시 검증한다.

이번 003 구현의 최종 검증 대상:

- [x] `npm run test -- --run`
- [x] `npm run lint`
- [x] `npm run typecheck`
- [x] `npm run build`
